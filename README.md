Reusable REST API contracts

## Setup

### Add Dependency

#### Gradle

```groovy
implementation("io.opengood.api:rest-api-contracts:VERSION")
```

#### Maven

```xml
<dependency>
    <groupId>io.opengood.api</groupId>
    <artifactId>rest-api-contracts</artifactId>
    <version>VERSION</version>
</dependency>
```
